Chiamak Key Layout

  • Hover over the [C] keys to read about combos.
  • Blank keys are transparent
  • Sticky mods in pink
  • Page credits to baobaozi https://codepen.io/baobaozi/pen/mdOjpQp

Typing

Alphas follow Chiamak. Uses combos, thumb shift, left space, right-thumb e.

Numbers/Symbols

Hold down the SYM key to activate this layer. Numpad on right hand, corresponding symbols on left, mirrored, with extras scattered around.

Navigation

Hold down the NAV key to activate this layer. Inverted-T arrows, common keyboard shortcuts, and tab/workspace navigation macros

Function

F1-12 on left, and media keys on right, as well as some mouse keys for use with my Charybdis Nano
